# Бодлого 7
Аркад тоглоомын тоглогч лидер самбарын хамгийн орой руу гарахыг хүсэж, байрлалаа тогтмол хянахыг хүсч байна. Тус тоглоом нь "Нягт эрэмбэлэлтийн систем" (Dense Ranking) ашигладаг ба самбарын ажиллах зарчим нь дараах байдалтай: 
Хамгийн өндөр оноотой тоглогч самбарын 1-р байрт орно.
Ижил оноотой тоглогчид ижил байр эзэлнэ. Дараагийн тоглогч дараалсан дараагийн байранд орно.
